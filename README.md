# python-web-flask-mvc-matplotlib-pyplot-bar-chart-image

## Description
A POC for showing matplotlib
chart in a flask mvc.

## Tech stack
- python
  - flask
  - matplotlib
  - pyplot

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- Available http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
